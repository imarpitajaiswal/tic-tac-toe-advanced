# Tic-Tac-Toe-Advanced

## 🎯 The Why
An interactive Tic-Tac-Toe game featuring real-time score tracking and persistent data storage. This project showcases the implementation of Python's `pickle` module for serializing game data.

## 🚀 Key Features
* **Persistence Layer**: Automatically saves player scores to a `scores.pkl` file so they persist between sessions.
* **OOP Architecture**: Built using Python classes to ensure clean, scalable, and modular code.
* **Dynamic UI**: Real-time score updating and interactive grid feedback.

## 🛠 Tech Stack
| Category | Technology |
| :--- | :--- |
| **Language** | Python (3.14) |
| **GUI** | `tkinter` |
| **Data Persistence** | `pickle` (Object serialization) |

## ⚙️ How to Run
1. `pip install tk` (if not installed)
2. `python main.py`
